# Intuit Build Challenge

This repository contains solutions to assignments part of the build challenge 

### Assignment 1: Producer-Consumer Pattern
- Implement producer-consumer pattern with thread synchronization

- [View Assignment 1 ](https://github.com/ijanhv/build_challenge/tree/main/producer-consumer)

### Assignment 2: Sales Data Analysis

- A data analysis application processing CSV sales data with Java Streams

- [View Assignment 2 ](https://github.com/ijanhv/build_challenge/tree/main/sales-analyzer) 



##  Tech Stack

- **Language**: Java 17+
- **Testing**: JUnit 5
- **Build Tool**: Maven/Gradle
- **IDE**: IntelliJ IDEA / Eclipse / VS Code

##  Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/ijanhv/build_challenge.git
   cd build_challenge
   ```

2. **Navigate to specific assignment**
   ```bash
   cd producer-consumer  # or sales-analyzer
   ```

3. **Follow individual README instructions** in each assignment folder

## Key Learning Objectives

- **Concurrent Programming**: Thread synchronization, inter-thread communication
- **Functional Programming**: Streams, lambdas, immutability
- **Data Processing**: CSV parsing, aggregation, statistical analysis
- **Clean Code**: Separation of concerns, testability, documentation
- **Testing**: Comprehensive unit tests with JUnit 5
